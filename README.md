## Run Locally

Clone the project

```bash
  git clone https://github.com/DevGar1/threejs-basics.git
```

Go to the project directory

```bash
  cd threejs-basics
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run build
```
Open index file with a extension as Live server

```bash
dist/index.html
```
